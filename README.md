# Medical Training App

AI-powered medical education app for medical students to practice diagnostic skills.

## 🏥 Features
- AI patient interviews with Claude
- Medical test ordering and results
- Bilingual support (English/Portuguese)
- Progress tracking and analytics
- PDF extraction pipeline for medical textbooks

## 📱 iOS App
SwiftUI app with Core Data for patient simulation and learning.

## 🐍 Python Scripts
- `pdf_extractor.py` - Extract medical conditions from PDFs
- `manual_curator.py` - Review and edit extracted data  
- `database_populator.py` - Populate app database

## 🚀 Setup
1. Add Claude API key to config
2. Run Python extraction pipeline
3. Open iOS app in Xcode
